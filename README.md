# Store Sales Dashboard Analysis

## Project Overview

Developed an Excel dashboard to analyze retail sales performance using Pivot Tables, Pivot Charts, and Slicers.

## Key Features

* Sales KPI Analysis
* Customer Demographics Analysis
* Regional Sales Performance
* Sales Channel Analysis
* Dashboard Filters

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Data Visualization

## Key Insights

* Identified top-performing sales channels and regions.
* Analyzed customer demographics and purchasing patterns.
* Generated actionable business insights through KPI reporting.
